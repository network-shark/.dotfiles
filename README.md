Keystrokes
### Navigate with FD and FZF

CTRL-T 	to fuzzily search for a file or directory in your home directory then insert its path at the cursor
ALT-C  	to fuzzily search for a directory in your home directory then cd into it
CTRL-R 	to fuzzily search your command line history then run an old command

### Tmux
###### create panes

###### vertical
CTRL + B + |
###### horizontal
CTRL + B + -

# resize panes
ALT + HJKL

# switch panes
CTRL + H J K L

# rename pane
CTRL + B + ,


# swap pane to pane -1
CTRL + B +  {
# swap pane to pane +1
CTRL + B + }
# all at once
CTRL + B + o

# split panes evenly
# vertical
CTRL + B + ALT + 1
# horizontal
CTRL + B + ALT + 2

# fzf and vim
# AG
CTRL + g
# Files
CTRL + p
# move up
CTRL + k
# down
CTRL + j

### VIM

###### horizontal VIM Pane
CTRL + W + s
###### vertical VIM Pane
CTRL + W + v
###### Switch VIM Pane
CTRL + W
